# Pleat Calculator

A handy app for calculating pleats with precision and style.

## Demo

![image](src/static/screenshots/pleatcalculator.web.app_.png "Demo")

## Development

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Firebase account](https://firebase.google.com/)

### Installation

```bash
npm install
```

### Build

```bash
npm run build
```

### Publish

```bash
firebase deploy
```
